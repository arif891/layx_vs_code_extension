# Layx VS Code Extension

## Features

- Syncs HTML elements with matching data-component attributes across files
- Creates backups of modified files (optional)
- Progress indication during sync
- Detailed logging

## Requirements

- VS Code ^1.96.0
- Files must use UTF-8 encoding

## Extension Settings

- `layx.createBackups`: Enable/disable backup file creation before syncing

## Usage

1. Open command palette (Ctrl/Cmd + Shift + P)
2. Type "Sync Element" and select the command
3. The extension will sync all matching elements across HTML files
